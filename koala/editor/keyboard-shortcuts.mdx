## Overview

Master keyboard shortcuts to significantly boost your productivity with Koala Data Explorer. All shortcuts are customizable through VS Code's keyboard shortcut editor.

## Essential Shortcuts

### Query Execution

| Action | Windows/Linux | Mac | Description |
|--------|--------------|-----|-------------|
| Execute Query | `Ctrl+Enter` | `Cmd+Enter` | Run entire query or current statement |
| Execute Selection | `Ctrl+Shift+Enter` | `Cmd+Shift+Enter` | Run only selected SQL text |
| Execute at Cursor | `Alt+Enter` | `Option+Enter` | Run statement at cursor position |
| Cancel Execution | `Ctrl+Alt+C` | `Cmd+Option+C` | Stop running query |
| Explain Plan | `Ctrl+E` | `Cmd+E` | Show execution plan (Paid) |

## Customizing Shortcuts

### Access Keyboard Shortcuts

1. Open Command Palette: `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (Mac)
2. Type "Preferences: Open Keyboard Shortcuts"
3. Search for "Koala" to find all extension shortcuts

### Modify Shortcuts

1. Click on any shortcut to edit
2. Press desired key combination
3. Press Enter to save

### Create Custom Shortcuts

Add custom shortcuts in `keybindings.json`:

```json
[
  {
    "key": "ctrl+shift+q",
    "command": "koala.executeQuery",
    "when": "editorTextFocus && editorLangId == sql"
  },
  {
    "key": "ctrl+shift+s",
    "command": "koala.saveResults",
    "when": "koala.resultsVisible"
  }
]
```

## Context-Specific Shortcuts

### When in SQL Editor

These shortcuts only work when focused on SQL editor:

- Format SQL: `Shift+Alt+F`
- Run Query: `Ctrl+Enter`
- Comment Line: `Ctrl+/`

### When in Results Panel

These shortcuts only work when focused on results:

- Copy Cell: `Ctrl+C`
- Export: `Ctrl+S`
- Filter: `Ctrl+F`

### When in Schema Explorer

These shortcuts only work in schema explorer:

- Expand Node: `Right Arrow`
- Collapse Node: `Left Arrow`
- Generate SELECT: `Ctrl+G`

## Productivity Tips

### Essential Combinations

Master these for maximum efficiency:

1. `Ctrl+Enter` → Execute query
2. `Ctrl+Space` → Trigger autocomplete
3. `Ctrl+/` → Comment/uncomment
4. `Shift+Alt+F` → Format SQL
5. `Ctrl+K, C` → Switch connections

### Workflow Shortcuts

Common workflow patterns:

1. **Quick Query**: `Ctrl+T` → Type SQL → `Ctrl+Enter`
2. **Export Results**: Execute → `Ctrl+R` → `Ctrl+Shift+E`
3. **History Recall**: `Ctrl+H` → Select → `Enter`
4. **Multi-Statement**: Select → `Ctrl+Shift+Enter`

## Conflict Resolution

### VS Code Conflicts

Some shortcuts may conflict with VS Code defaults. To resolve:

1. Identify conflict in Keyboard Shortcuts
2. Decide which command takes priority
3. Reassign one of the shortcuts
4. Test both commands work

### Common Conflicts

- `Ctrl+K`: VS Code uses for many commands
- `Ctrl+Space`: May conflict with system shortcuts
- `F5`: Conflicts with VS Code debug

## Platform Differences

### Windows vs Mac

Key mapping differences:

- `Ctrl` → `Cmd`
- `Alt` → `Option`
- `Enter` → `Return`
- `Backspace` → `Delete`

### Linux Considerations

Some shortcuts may conflict with window manager:

- `Alt+Click`: Often used for window dragging
- `Ctrl+Alt+Arrow`: Virtual desktop switching
- Configure in system settings if needed
