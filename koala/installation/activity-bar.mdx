## Quick access

Koala adds a dedicated icon to the VS Code Activity Bar (left sidebar) so you can open the full experience with one click.

## Open Koala from the Activity Bar

1. Click the **Koala** icon in the Activity Bar.
2. The Koala panel opens with the query editor at the top and results pane below.
3. Use the left sidebar to switch between **Connections**, **Data Sources**, and recent activity.

<Tip>
  You can drag the Koala icon to reorder it in the Activity Bar if you use it frequently.
</Tip>

## Panel layout

- **Query editor**: Monaco-based SQL editor with execution controls.
- **Results grid**: Interactive grid with sorting, filtering, and export actions.
- **Data sources sidebar**: Table and view explorer for connected environments.

<Note>
  The Activity Bar entry appears after installation. If you do not see it, ensure the extension is enabled for the workspace.
</Note>
