## Supported Platforms

- **macOS:** Monterey (12) or later on Intel or Apple Silicon.
- **Windows:** Windows 10 (21H2) or Windows 11 with the WebView2 runtime.

## Account Access

- A Koala account (same credentials you use in the VS Code extension) to sign in via Koala Cloud.
- Network access to your Oracle Fusion OTBI/BIP environments.
- Optional: BTSS license key if you plan to unlock paid features.
  - License tiers and activation flow match the VS Code extension (`koala/licensing/*`).

## Hardware & Permissions

- 8 GB RAM minimum recommended for handling multiple result sets.
- ~1 GB free disk space for the application plus additional space for cached query results.
- Ability to install signed applications (admin rights may be required on locked-down machines).
