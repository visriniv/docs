---
title: "Free vs Paid Features"
description: "Understand what’s included in each Koala Studio tier"
---

## Free Tier

Ideal for light usage or evaluation.

- Single query tab
- Basic authentication
- 50 rows per query limit
- CSV/JSON export
- SQL history capped at 50 entries

## Paid License

Unlock the full desktop experience.

- Unlimited query tabs and saved queries
- OAuth PKCE + Windows SSO authentication
- Unlimited rows per query
- Excel export
- Data sources explorer
- Performance monitor
- Extended history (up to 500 entries)

Licenses purchased for the VS Code extension also apply to Koala Studio—activate once and use both products interchangeably.
