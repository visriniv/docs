## Layout Overview

1. **Top bar** – New Tab, connection picker, Run/Stop buttons, export shortcuts, profile avatar.
2. **Tabs row** – rename, drag to reorder, duplicate using the context menu.
3. **Main Monaco editor** – syntax highlighting, minimap, gutter breakpoints, and inline error messages.
4. **Result pane** – appears after execution with tabs for Grid, JSON, or Messages.
5. **Right rail** – optional SQL history, saved queries, and tab metadata.
6. **Left explorer** – data sources browser, quick links to query history and connections.

## Customizing the View

- Toggle the explorer or right rail using their icons on the left.
- Drag the divider between editor and results to resize.
- Switch to light/dark mode from the profile menu; Koala remembers per device.

## Tab Controls

- **Double-click** a tab name to rename.
- **Ctrl/Cmd+W** closes the tab.
- Right-click for options such as Duplicate, Close Others, or Copy full SQL.

## Connection Picker

- Located above the editor; shows the currently selected connection.
- Clicking opens the dropdown with search and favorites, matching what you set in the Connections drawer.
