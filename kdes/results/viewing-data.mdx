## Grid Basics

- After running a query, results appear in the lower paneâ€™s **Grid** tab.
- Columns auto-size to content; double-click a header edge to fit.
- Hover to reveal tooltips for truncated text or numbers.

## Pagination

- Large result sets are broken into 1,000-row pages.
- Use the pagination controls in the top-right corner of the grid to move between pages without rerunning the query.

## Columns Panel

- Click the column chooser icon to hide/show columns.
- Drag headers to reorder columns; Koala remembers per tab.

## Messages & JSON Views

- Switch to the **Messages** tab to see Oracle warnings/errors.
- Choose **JSON** if your query returns JSON output or if you used the JSON export option.
