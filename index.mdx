## Query Oracle Fusion from Claude

ORCA puts our consultants' Oracle expertise on tap. Enable its **governed MCP connector** (Model Context Protocol) and any **MCP client** — an AI assistant like Claude Desktop or Claude Code — can query your Oracle Fusion data in plain language, or even build a **live dashboard** over it. Every request runs within your own Oracle security, and the connector keeps getting smarter as we ship new skills.
